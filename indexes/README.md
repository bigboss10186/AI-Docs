# Knowledge Indexes

这里放跨主题索引。索引负责回答“已有知识放在哪里、相关概念如何连接、同类问题如何归档”。

## 索引列表

- [数据库内核索引](database.md): PostgreSQL、GaussDB/openGauss、存储、WAL/replay 和诊断相关笔记。
- [AI / Codex 索引](ai-codex.md): Codex 源码、agent loop、工具执行和上下文管理相关笔记。
- [源码阅读索引](source-reading.md): 适用于 PostgreSQL、openGauss 和 Codex 的源码阅读笔记归档。

## 使用方式

1. 新增专题笔记后，在相关索引里补一个链接。
2. 一个主题横跨多个目录时，只在索引里建立关联，不必移动原文。
3. `inbox/` 中的临时问题整理后，可以转成正式笔记，也可以只在索引里记录归属。
