# Algorithms

这个目录用于归档算法、数据结构和工程复杂度优化相关知识。

## 范围

- `complexity-saving-patterns/`: 复杂度节省技巧和工程实践。
- `data-structures/`: 数据结构选择、适用场景和取舍。
- `dynamic-programming/`: 状态设计、转移优化和常见模型。
- `graph/`: 图搜索、最短路、连通性和拓扑关系。
- `string/`: 字符串匹配、哈希、字典树和自动机。

## 已归档笔记

- [复杂度节省技巧](complexity-saving-patterns/complexity-saving-patterns.md): 总结工程和算法中常见的复杂度优化思路，重点关注如何用更合适的数据结构、状态表示和计算方式减少时间或空间开销。

## 归档规则

1. 概念类内容按算法主题归档。
2. 工程实践类内容优先记录问题背景、复杂度瓶颈、优化方式和适用条件。
3. 如果一个技巧横跨多个主题，可以在 `../indexes/` 中建立关联索引。
